# MV Tracker

일간/주간 라이프 트래커 — 몸 만들기, 체력관리, 작업 속도, 에너지 관리

## 기능
- **몸 만들기** — 운동 체크 (웨이트/유산소/스트레칭/걷기), 체중, 수면
- **체력 관리** — 식단, 단백질, 수분 섭취 체크
- **작업 현황** — 딥워크/촬영/편집/기획 체크 + 일일 작업 시간
- **에너지** — 에너지 레벨(1-5), 오늘의 한 줄 메모
- **주간 리뷰** — 항목별 달성률, 작업 시간 합계, 메모 모아보기
- **LocalStorage 저장** — 데이터는 브라우저에 영구 보관

## 실행

```bash
npm install
npm run dev
```

## 기술 스택
- React 18 + Vite
- LocalStorage persistence
- No external UI library
